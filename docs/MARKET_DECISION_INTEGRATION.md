# Market Decision Integration

The Decision Engine now enriches recommendations with stored market insight context.

This integration is recommendation-only:

- no Ozon API execution;
- no price changes;
- no bid changes;
- no stock changes;
- no Telegram integration;
- no Approval Workflow changes.

## Data Flow

```text
Internal Metrics
+ Market Insights
-> Market Context
-> Decision Engine
-> Recommendation
```

## Market Context

`src/ozon_agent/decision/market_context.py` maps saved market insights into:

- `price_pressure`
- `competitor_growth`
- `review_pressure`
- `rating_pressure`
- `market_risk_score`
- `market_opportunity_score`

It also exposes:

- `market_signals`
- `market_risks`
- `market_opportunities`

## Mapping

- `PRICE_DROP` -> high price pressure when score/count is high.
- `REVIEW_SURGE` -> review pressure.
- `NEW_COMPETITOR` -> competitor growth.
- `ASSORTMENT_GAP` -> market opportunity.

## Recommendation Enrichment

Generated recommendations now include:

- `market_signals`
- `market_risks`
- `market_opportunities`
- `supporting_metrics.market_context`

The recommendation action remains generated by the existing Decision Engine rules. Market context
adds explanation and supporting evidence; it does not execute any action.

## CLI

```bash
ozon-agent recommendations market-context
ozon-agent recommendations market-context --sku SKU123
ozon-agent recommendations explain
ozon-agent recommendations explain --sku SKU123 --top 5
```

`recommendations explain` shows the internal recommendation reason plus market signals, risks, and
opportunities.
